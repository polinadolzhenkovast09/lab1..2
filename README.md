## Рассчитать значение f при заданном значении вещественного числа x: При выводе на экран оставьте 2 знака после запятой.
````
# Выполнил: Долженкова Полина Викторовна
# Группа: АБП - 231


x = float(input())
if x >= 0:
    f = x**0.5 + x **2
else:
    f = 1 / x

print(round(f,2))
````
